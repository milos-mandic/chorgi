# Scheduling Preferences

## Working Hours
- Weekdays: 9:00 AM - 6:00 PM
- Weekends: Off (no scheduling unless explicitly requested)

## Task Type Preferences
- Deep work / coding / creative: Morning (9:00 AM - 12:00 PM)
- Meetings / calls: Afternoon (1:00 PM - 4:00 PM)
- Admin / emails / light tasks: End of day (4:00 PM - 6:00 PM)
- Exercise / personal: Early morning or evening (outside work hours)

## Defaults
- Default task duration: 60 minutes
- Buffer between events: 15 minutes
- Default importance: medium
- Scheduling horizon (no date given): next 14 days
